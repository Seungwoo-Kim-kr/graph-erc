# Data Directory

Raw datasets, processed files, and cached features are **not tracked by git** (see `.gitignore`).
Use the download script to populate `raw/` automatically:

```bash
bash scripts/download_data.sh
```

## Expected structure

```
data/
├── raw/
│   ├── meld/
│   │   ├── train_sent_emo.csv
│   │   ├── dev_sent_emo.csv
│   │   └── test_sent_emo.csv
│   └── emorynlp/
│       ├── train.json
│       ├── dev.json
│       └── test.json
│
├── processed/          # generated by run_experiment.py
│   └── meld/
│       ├── train_dialogues.json
│       ├── dev_dialogues.json
│       └── test_dialogues.json
│
├── graphs/             # generated by run_experiment.py (NetworkX pickles)
│   └── meld/
│       ├── train_graphs.pkl
│       ├── dev_graphs.pkl
│       └── test_graphs.pkl
│
└── audio_cache/        # generated on first run (HuBERT embeddings)
    └── meld_test_audio.pkl
```

## Notes

- `processed/` and `graphs/` are created automatically on the first `run_experiment.py` run.
- `audio_cache/` requires a GPU or sufficient RAM for HuBERT inference; set `use_audio: false` in `src/config.py` to skip it.
- Total disk usage after full preprocessing: ~500 MB (dominated by audio cache).
